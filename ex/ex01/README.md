(テーマ)コンパイル環境確認プログラム
========================================

実習内容
--------

ファイルをsandbox以下にコピーしてmakeする。

    % cd ~/daqmw-tc/sandbox
    % cp -r ../ex/ex1 . (最後にカレントディレクトリを示すドット(.)があることに注意)
    % cd ex1
    % make

確認
----

- エラーがでない
- sample という実行ファイルができていることを確認する

実行方法
--------

    % ./sample

とするとhello, worldと画面に表示されるのを確認する。
